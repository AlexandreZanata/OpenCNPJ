# Commit Convention

Formato:

`type(scope): subject`

Tipos permitidos:
- feat
- fix
- perf
- refactor
- test
- bench
- docs
- ci
- chore
- build
- revert

Exemplos:
- `feat(parser): add Latin-1 to UTF-8 conversion in CSV reader`
- `perf(loader): replace batch insert with COPY FROM STDIN`
- `test(parser): cover date null edge case`
